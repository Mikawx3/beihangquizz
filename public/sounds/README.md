# Sons pour l'animation

Pour ajouter le son de roulement de tambour à l'animation de révélation :

1. Placez un fichier audio nommé `drum-roll.mp3` dans ce dossier (`public/sounds/`)
2. Le fichier sera automatiquement chargé et joué pendant l'animation

## Format recommandé
- Format : MP3
- Durée : 3-5 secondes (le son sera en boucle pendant l'animation)
- Volume : Le volume sera ajusté automatiquement à 70%

## Sources de sons libres
Vous pouvez trouver des sons de roulement de tambour gratuits sur :
- Freesound.org
- Zapsplat.com
- Pixabay.com (section audio)
